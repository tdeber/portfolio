# Tetris en C++

Proyecto desarrollado en el marco de la materia Programación Orientada a Objetos.

El objetivo del trabajo consistía en desarrollar una aplicación que permitiera aplicar los conceptos fundamentales de la programación orientada a objetos.
Como propuesta se podía optar por la implementación de un pequeño juego o un sistema de gestión. En este caso se eligió desarrollar el clásico juego Tetris.

Para la parte gráfica se utilizó la biblioteca **SFML**, la cual fue investigada e incorporada de manera autónoma para poder cumplir con los requisitos del proyecto.

El desarrollo se realizó utilizando el entorno **Zinjai**.

---

## Características

- Implementación del juego Tetris
- Manejo de piezas, rotaciones y detección de colisiones
- Sistema básico de puntaje
- Lógica de caída progresiva
- Renderizado gráfico mediante SFML
- Organización del código siguiendo principios de POO

---

## Conceptos aplicados

- Programación Orientada a Objetos
- Encapsulamiento y modularidad
- Manejo de eventos por teclado
- Lógica de actualización en tiempo real
- Integración de bibliotecas externas

---

## Tecnologías y herramientas

- C++
- SFML
- Zinjai
